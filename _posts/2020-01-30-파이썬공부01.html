파이썬으로 게시판만들기 01

가상환경만들기 / 순서적으로 정리


가상환경이 될 디렉터리를 만든다

mkdir day2

# day2라는 이름의 폴더가 생성된다.


가상환경을 생성한다

pyenv virtualenv 3.6.2 day2

# day2이름의 폴더에 3.6.2버전의 python 가상환경을 생성한다.


가상환경에 django를 설치한다.

pip install django==2.0.2

# 2.0.2버전의 django를 설치한다.


admin project 생성

django-admin startproject config

# config이름의 프로젝트가 생성된다.


관리의 편의를 위해 project명 변경

mv config django_app2

# django_app2라는 프로젝트명으로 변경


내가 만든 가상환경중 선택을 한다

pyenv local (탭)

# 만든 가상환경들의 리스트들을 볼 수 있다.


이후 작업

python manage.py makemigrations

python manage.py migrate

관리자가 필요할 시

python manage.py createsuperuser

로컬서버 실행하기

python manage.py runserver



여기까지가 프로젝트 생성에 대한 기본

기본적으로 파이썬에서 제공한 django와 연결되어 있으며 db를 변경하고 싶으면 setting.py에서 변경하면된다.

잘 생성이 되어 있다면 127.0.0.1:8000 으로 들어가면 파이썬이 연결 된 페이지를 확인할 수 있다.
