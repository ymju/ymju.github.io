파이썬으로 게시판만들기 02

MVT Pattern

Model, View, Templates로 나뉘어 관리한다.


Model / Model.py

Class Board는 데이터베이스의 테이블에 해당되며, 안에 속성(subject, content)들은 컬럼이 된다.


View / views.py

URL 요청이 들어오면 Views.py에 정의된 해당 URL을 찾아 실행한다.


templates / 별도의 폴더로 관리

UI를 담당하게 된다.

html의 경우 내부에서 django에서 제공하는 별도의 language를 사용한다.
